# ScriptMarketGAG

Koleksi script Roblox untuk **Grow a Garden: Trade World**.

## Isi

| File | Deskripsi |
|------|-----------|
| `GAGSniper.lua` | Pet sniper multi-profil (tab Buy + Misc), server-hop, Discord webhook. |
| `allegiaanthub.lua` | Versi lanjutan GAGSniper dengan **join-by-pet** via Global Index (`TokenRAPs.FindSellers` + `TeleportToListing`). |
| `GAGSeller.lua` | Auto jualan: **Auto Claim Booth** + auto-list pet dari inventory (5 profil: Pet Types, Mutation, Min/Max Weight, Max Listings, Price). |
| `CoastalSniper.lua` | Sniper varian Coastal. |

## Pakai

```lua
loadstring(readfile("allegiaanthub.lua"))()
```
